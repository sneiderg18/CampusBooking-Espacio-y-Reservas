#  CampusBooking Lite – Gestión de Espacios y Reservas

**CampusBooking Lite** es una aplicación web desarrollada en **Laravel** para la administración de **espacios** (auditorios, salones, laboratorios, etc.) y la gestión de **reservas** realizadas por los usuarios del campus.  
Permite crear, editar, listar y eliminar espacios y reservas, con validaciones completas y diseño unificado.

---

##  Características principales

###  Módulo de Espacios
- Crear nuevos espacios del campus.
- Editar información existente.
- Eliminar espacios con confirmación (SweetAlert2).
- Validaciones con mensajes personalizados.
- Listado en tabla.

###  Módulo de Reservas
- Crear reservas vinculadas a un espacio.
- Validaciones completas para fecha, horas y campos requeridos.
- Confirmación antes de eliminar.
- Alertas uniformes con el módulo de espacios.
